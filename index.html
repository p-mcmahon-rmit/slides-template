What is coding?

The process of designing, writing, testing, and maintaining the source code of computer programs.

It involves using programming languages to create instructions that computers interpret and execute to perform specific tasks.

Interaction Design

Code can be used to craft a set of instructions that produce interactive creative works.

Just as you might use other design tools to create images, layouts or write text, programmers use coding languages to define dynamic system behaviour.

Why should I learn to code?

A basic understanding of coding concepts can be highly beneficial in today’s multidisciplinary design context.

It fosters collaboration, efficiency, and allows you to creative more distinctive & customised digital works.

Why does it matter?

Allows you to build whole projects from ground up.

Gives deeper insight and skills transferable to other digital tools.

Encourages critical engagement with computational culture.

Why web code?

Drawing on our personal experience and expertise.

Simple and well documented entry point.

Affords design excellence through both simplicity and complexity.
